## 🚀 如何运行

### 方式 1：浏览器调试（推荐先用这个）
```
1. 用任意本地服务器打开 minigame/index.html
   例如：python -m http.server 8080
2. 访问 http://localhost:8080/minigame/index.html
```

### 方式 2：微信开发者工具（最终目标）
```
1. 下载微信开发者工具（Stable 版）
2. 打开开发者工具 → 导入项目
3. 选择目录：d:/work/games/minigame
4. 填入 AppID（测试号或正式号）
5. 即可预览
```

### 重新生成关卡
```bash
cd d:/work/games/tools
node build-levels.js
```
